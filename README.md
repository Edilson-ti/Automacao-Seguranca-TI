# Automação e Segurança em Ambiente Windows (PowerShell)

Scripts desenvolvidos para automação de rotinas de segurança e monitoramento em ambientes corporativos Windows.

## 🎯 Objetivo
Implementar controles proativos para detecção de atividades suspeitas, auditoria de eventos e geração de relatórios automatizados.

## 🔐 Funcionalidades

- Detecção de exclusão em massa de arquivos
- Geração automática de relatórios de auditoria
- Monitoramento de eventos críticos do sistema

## 🛠 Tecnologias utilizadas
- PowerShell
- Windows Server
- Event Viewer (Logs do Windows)
- Auditoria NTFS

## 💼 Cenário de uso
Ambientes corporativos que necessitam:
- Monitoramento de segurança
- Prevenção de incidentes (ex: ransomware)
- Auditoria e rastreabilidade de dados
